Simple Game Engine
------------------
This program runs a simple RPG-style game by using the data from
the files in the data/ directory.

Simple Game Creator
-------------------
This program allows you to modify the original player stats, rooms,
monsters, and shops of the game, in order to create your own simple
RPG game.
